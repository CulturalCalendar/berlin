About this calendar

This is a personal project aggregating cultural events happening in Berlin — concerts, exhibitions, theater, parties, film screenings, readings, and more.

Events are scraped weekly from a range of Berlin cultural publications and venue websites, then processed by an AI model (Google Gemini) which extracts event details, tags, and writes them into a shared Google Calendar. The website reads directly from that calendar in real time.

Sources include: Siegessäule, The Next Day Berlin, CeeCee, Index Berlin, Tip Berlin, The Berliner, Resident Advisor, Volksbühne, Maxim Gorki Theater, Berghain, Schinkelpavillon, Fotografiska, We Are Village, and others.

The project is built with Google Apps Script, the Gemini API, and plain HTML/JS — no backend, no database, no tracking.

Made by Tom Offrede.
